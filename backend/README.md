## Dev notes - Backend

### postgres

    sudo apt-get install postgresql postgresql-contrib
    sudo -u postgres createuser --superuser jdisctf
    sudo su - postgres
    createdb -O jdisctf jdisctf_db

### venv

    pip install virtualenv
    virtualenv venv
    source venv/bin/activate
